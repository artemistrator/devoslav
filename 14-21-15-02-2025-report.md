## 🎨 Задача 2: Загрузка изображений с AI-анализом (Vision)

### Проблема
Пользователю нужно было прикрепить макет/скриншот к задаче, чтобы система автоматически распознала изображение через OpenAI Vision и сохранила текстовое описание для разработчика.

### Требования
1. **Database (Prisma):**
   - Создать модель `TaskAttachment`
   - Поля: id, taskId, filePath, fileName, mimeType, visionAnalysis, createdAt
   - Добавить связь в модель `Task`

2. **Backend (API):**
   - Роут `/api/upload/task-attachment`
   - Сохранить файл локально в `public/uploads`
   - Отправить файл в OpenAI Vision API (gpt-4o)
   - Промпт: "Analyze this UI screenshot. Describe layout, colors, typography, component placement, spacing, and any specific design details for a frontend developer. Be technical and precise. Include specific color codes if visible, mention dimensions/spacing ratios, and describe any interactive elements."
   - Сохранить результат анализа в `visionAnalysis`

3. **Frontend (Task Details):**
   - Вкладка "Design / Attachments" в модалке задачи
   - Drag & Drop зона или кнопка "Upload Image"
   - Превью картинки
   - Аккордеон "AI Design Analysis" с возможностью редактирования

4. **Integration (Prompt Generator):**
   - Обновить `lib/agents/prompt-generator.ts`
   - Добавить visionAnalysis в промпт для кодинга

**Именование файлов:**

- Сохранён: `timestamp_sanitizedName`
- Пример: `1739641234567_screenshot.png`

**OpenAI Vision интеграция:**
```javascript
const visionResponse = await openai.chat.completions.create({
  model: "gpt-4o",
  messages: [
    {
      role: "user",
      content: [
        {
          type: "text",
          text: "Analyze this UI screenshot. Describe layout, colors, typography, component placement, spacing, and any specific design details for a frontend developer. Be technical and precise. Include specific color codes if visible, mention dimensions/spacing ratios, and describe any interactive elements.",
        },
        {
          type: "image_url",
          image_url: { url: `data:${mimeType};base64,${base64}` },
        },
      ],
    },
  ],
  max_tokens: 2000,
});
```

**Генерация промпта с контекстом дизайна:**
```typescript
const visionContexts = task.attachments
  .filter(a => a.visionAnalysis?.trim().length > 0)
  .map(a => `
    === DESIGN IMAGE: ${a.fileName} ===
    Image URL: ${a.filePath}
    
    AI Vision Analysis:
    ${a.visionAnalysis}
  `)
  .join("\n");

const visionInstruction = visionContexts.length > 0
  ? `
    === DESIGN CONTEXT FROM IMAGES ===
    ${visionContexts}
    
    IMPORTANT: The AI Vision Analysis above describes design. Use these details to implement UI/layout correctly.
  `
  : "";
```

**Трекинг использования:**
```typescript
await trackAIUsage(
  {
    usage: {
      promptTokens: visionResponse.usage.prompt_tokens,
      completionTokens: visionResponse.usage.completion_tokens,
      totalTokens: visionResponse.usage.total_tokens,
    },
    model: "gpt-4o",
  },
  {
    projectId: task.planId,
    taskId: task.id,
    actionType: "vision_analysis",
    model: "gpt-4o",
  }
);
```
